What the driver gets

Same content as today's per-fridge PDF sheet
  • Products in category print order (rule R8) so the sheet matches the fridge shelf layout.
  • Check-off replaces the paper In-box / Missing columns; misses feed the reconciliation view automatically.
  • Withdrawal list = products in the fridge whose DLC expires before the next delivery — pulled, counted, logged.
  • Works read-only offline; check-offs sync when back online.
Until this view ships, the same data is still emailed as PDF/XLSX sheets on “Confirm dispatch” — the mobile view is additive, not a replacement.
